Creates a motor driver by initializing GPIO pins and turning off the motor for safety.

Parameters
en_pinThe value sent to the enable pin controlled by the microcontroller. When the en_pin is set high, it enables the motor.
in1pwmThe initial pwm percentage value sent to the timer channel corresponding to the first motor pin. When in1pwm is set to 0, the motor runs in the forward direction.
in2pwmThe initial pwm percentage value sent to the timer channel corresponding to the second motor pin. When in2pwm is set to 0, the motor runs in the reverse direction.
timerThe timer channel that corresponds with the motor pins used for the motor.
enapinThe chosen enable pin number controlled by the microcontroller corresponding to EN_A/OCD_A or EN_B/OCD_B on the L6206.
motorpin1The first chosen motor pin on the CPU corresponding to the IN1 A/B pin on the L6206.
motorpin2The second chosen motor pin on the CPU corresponding to the IN2_A/B pin on the L6206.
